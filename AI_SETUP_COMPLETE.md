## ğŸ› Troubleshooting

### AI Generates Incorrect Code

**Solution:** Remind AI of the rules
```
"Remember to follow BMAD_RULES.md:
- Use glassmorphism UI
- Add haptic feedback
- Support themes
Check .context/patterns.md"
```

### AI Doesn't Follow Patterns

**Solution:** Reference specific templates
```
"Use the exact template from .context/examples/glassmorphism-component.tsx
and customize only [specific parts]"
```

### AI Mixes App Code

**Solution:** Specify app context
```
"This is for apps/happiness-app only.
Do not import from other apps.
Check BMAD_RULES.md for monorepo boundaries."
```

### AI Uses Wrong Imports

**Solution:** Specify import rules
```
"Always use @ alias for imports:
import { useChat } from '@/hooks/useChat';
Never use relative imports."
```
